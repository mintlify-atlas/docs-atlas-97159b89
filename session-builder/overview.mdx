The **Session Builder** is the first screen you see when opening Appium Inspector. From here you configure everything needed to start a new Appium session: the server to connect to, the capabilities that describe your target device and application, and optionally a saved set of capabilities for reuse.

The screen is organized into two main areas:

- **Server Details** — at the top, used to configure how the Inspector connects to an Appium server
- **Tab panel** — the main body of the screen, with three tabs for different workflows

## Sections

<CardGroup cols={2}>
  <Card title="Server Details" icon="server" href="/session-builder/server-details">

## App Settings

The application settings panel is accessible via the button in the top-right corner of the Session Builder screen. From there you can change the **theme** (light, dark, or system default) and the **language** used throughout the Inspector interface.

For full details, see the [Application Settings](/app-settings) page.
